# Farmalisto_Analytics_TFM
Este repositorio contiene los scripts desarrollados para el Trabajo Final de Máster orientado al diseño e implantación de un sistema analítico empresarial en un entorno e-commerce.

El sistema implementa un pipeline reproducible que integra:

- Construcción del dataset analítico.
- Análisis exploratorio.
- Modelos predictivos.
- Segmentación de clientes.
- Técnicas de data mining.

---

## Estructura del proyecto

01_build_dataset.py  
Construcción del dataset procesado y generación de variables derivadas.

02_eda.py  
Análisis exploratorio de datos y generación de visualizaciones operativas.

03_forecasting.py  
Modelo de forecasting de demanda.

04_logistic_model.py  
Modelo de predicción logística de riesgo de retraso.

05_segmentation.py  
Segmentación comportamental de clientes.

06_data_mining.py  
Aplicación de técnicas de minería de datos (afinidad, cohorts, ABC).

---

Ejecutar los scripts en orden:
